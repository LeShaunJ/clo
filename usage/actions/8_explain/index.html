    Explain

    Display documentation on a specified topic.

    clo explain clo explain [--verbose] [--help] {models,domains,logic,fields}

    Positional Arguments

    {models,domains,logic,fields}

    (REQUIRED) A topic to get further explanation on.

    Choices:

    • "models"
    • "domains"
    • "logic"
    • "fields"

    Options

    ‑‑verbose

    ‑v

    Display more details.

    Default: false

    Peripherals

    Supportive, non-operational functions.

    ‑‑help

    ‑h

    Show this help message and exit.
